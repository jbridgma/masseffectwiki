Compatibilities[]
Guns - This augmentation can be applied to gun-type weapons.
Description[]
Changes weapon fire to automatic.

+50% Rate of Fire
+20% Clip Size
-40% Weapon Damage
-40% Weapon Accuracy
Mechanics[]
Strategies[]
Restrictions[]
The following weapons are unable to use this augmentation:

Single-shot weapons:
N7 Piranha
Widow
Semi-automatic weapons:
Lanat
N7 Eagle
Ushior
Venom
Zalkin
All automatic weapons
Bugs[]
The game incorrectly flags the Pathfinder Pioneer as fully automatic, preventing this augmentation from being applied to it.
The two automatic weapons below are bugged and allow this augmentation to be used. Using this augmentation wastes an augmentation slot and there is no benefit to using this augmentation with these weapons.

Pathfinder Ranger
Pathfinder Deep Impact
